Mockup 6: Views Panel — Block Assignment & Multi-View Layout

What this mockup shows

  • Scene A: Views panel with existing views (desktop) — view cards at top with block lists and reorder controls, unassigned blocks section below. Delete button hidden when only 1 view exists. A block can appear in multiple views.
  • Scene B: "New View" form open — name field + chip-based block selector. Blocks are optional (unlike requirements in the Blocks panel). Shows duplicate name validation message.
  • Scene C: Narrow/mobile accordion view — same panel content inside an accordion section, with always-visible reorder buttons.
